# `repostack whoami`

显示当前选中的用户。

```bash
repostack whoami
```

## 执行流程

1. **加载基础配置**
   - 读取 `repostack.yaml`。

2. **读取 `.repostackrc`**
   - 解析文件内容，提取 `user=...` 的值。

3. **输出**
   - 如果已选择用户：打印用户名。
   - 如果未选择用户：打印 `(no user selected)`（灰色提示）。

## 涉及文件

| 文件 | 操作 |
|------|------|
| `repostack.yaml` | 读取 |
| `.repostackrc` | 读取 |

## 错误与边界情况

- 如果 `repostack.yaml` 不存在或解析失败，错误会向上抛出。
- `.repostackrc` 不存在时不会报错，仅显示 `(no user selected)`。

## 示例

```bash
repostack whoami
# 输出: alice

repostack whoami
# 输出: (no user selected)
```
